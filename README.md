# fraud-detection-ml-dl

This project detects fraudulent transactions in E-commerce using Machine Learning and Deep Learning models.

## Features
- Random Forest
- Neural Network (ANN)
- LSTM
- Streamlit UI

## Run
pip install -r requirements.txt
streamlit run app.py

## Live Demo
https://fraud-detection-ml-dl-n5iwvq8scj9eal3hcmrvs7.streamlit.app/

This project is a web-based Fraud Detection System built using Machine Learning and Deep Learning models. It analyzes transaction data in real-time and classifies whether a transaction is fraudulent or legitimate using an interactive Streamlit interface.
